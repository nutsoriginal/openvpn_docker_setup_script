# Quick OpenVPN setup script

1. Copy script to your debian/ubuntu server
2. Run openvpn_docker_setup.sh with sudo
3. Follow instructions
4. Enter and save your new CA Key passphrase (several times)
5. Enter your server IP or hostname
6. Enter your new client_name
7. Download generated /opt/openvpn/<client_name>.ovpn file
8. Setup OpenVPN Connect client with your <client_name>.ovpn file
9. Enjoy
